# Example output

What one full YouTube run produces, so you can judge the output without setting up keys and yt-dlp.

- `the_brief_example.md`: the audience analysis from `process_brief.js`
- `investment_brief_example.md`: the final dossier from `process_logic.js`, with the code-computed financial block

The creator here ("@LiftWithMara") is fictional. All quotes, metrics, and numbers are invented for illustration, but the numbers are internally consistent with the shipped `Fitness & Health` benchmark: core audience 18,400 views, $150 ticket, 0.3% / 1% conversion, so $8,280 conservative and $27,600 base.
